# Tekkom

Aplikasi berbasis Laravel yang dikembangkan sebagai project pembelajaran dan pengembangan web.

## 📌 Tentang Project

Tekkom merupakan aplikasi yang dibuat menggunakan **Laravel** dengan tujuan mempermudah pengelolaan data dan proses yang dibutuhkan dalam project.

## ⚙️ Instalasi

Clone repository:

```bash
git clone https://github.com/Mexami1/Tekkom.git
```

Masuk ke folder project:

```bash
cd Tekkom
```

Install dependency:

```bash
composer install
npm install
```

Salin file environment:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Jalankan migration:

```bash
php artisan migrate
```

Jalankan server Laravel:

```bash
php artisan serve
```
